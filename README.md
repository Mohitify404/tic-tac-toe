🎮 Tic Tac Toe

A simple and interactive Tic Tac Toe game built using HTML, CSS, and JavaScript. Play against a friend and enjoy a clean, responsive UI.

---

🚀 Features

- 🟦 2-player game (X vs O)
- ⚡ Real-time move updates
- 🏆 Winner detection logic
- 🔁 Restart game functionality
- 🎨 Clean and minimal UI
- 📱 Fully responsive design

---

🛠 Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla)

---

🧠 Game Logic

- The board is represented using a grid (3x3)
- Players take turns placing X and O
- Winning conditions are checked after every move:
  - Rows
  - Columns
  - Diagonals
- If all cells are filled and no winner → Draw
